# Python-Port-Scanner
This is a simple TCP port scanner written with Python.
### What it can do:
- Scans ports on a targets IP
- Detects open/closed ports
- Uses Pythons built in sockets
## Run using, python port_scanner.py
## Example Output:
- Enter IP: 122.43.239
- Port 8000 is open
### REMEMBER ONLY USE THIS EDUCATIONALLY. Do NOT scan networks you do not own or have no permission to test.
